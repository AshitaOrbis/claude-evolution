# Genealogy Citation Formats Reference

Proper source citation is essential for credible genealogy research. This guide follows the Evidence Explained standard.

## Why Citations Matter

1. **Credibility**: Others can verify your work
2. **Reproducibility**: You can find the source again
3. **Analysis**: Different sources have different reliability
4. **Collaboration**: Others can build on your research

## Citation Components

Every citation should include:

1. **Author/Creator** (who created the record)
2. **Title** (what the record is called)
3. **Publication Details** (where and when published)
4. **Location** (where the record is held)
5. **Specific Locator** (page, entry, frame number)
6. **Access Information** (how and when you accessed it)

## Template Formats

### Online Database Record

```
[Database name], database with images, [Website name]
(URL : accessed [date]), entry for [person name],
[specific locator information].
```

**Example**:
```
"1880 United States Federal Census," database with images,
Ancestry (https://www.ancestry.com : accessed 15 Dec 2025),
entry for Johann Mueller household, Philadelphia Ward 7,
Philadelphia, Pennsylvania; citing NARA microfilm T9, roll 1167.
```

### Church Record (Online)

```
[Parish name], [Record type], [date range]; [specific entry info];
[archive/repository]; digital images, [website] (URL : accessed [date]).
```

**Example**:
```
St. Michael's Catholic Church (Munich, Bavaria), Baptisms 1840-1860,
entry for Johann Mueller, 15 March 1850, p. 127;
Archdiocese of Munich Archives; digital images,
Matricula (https://data.matricula-online.eu : accessed 15 Dec 2025).
```

### Census Record

```
[Year] [Country] Census, [jurisdiction hierarchy],
[specific locator]; [archive], [microfilm/digital reference];
[access method and date].
```

**Example**:
```
1880 U.S. Census, Pennsylvania, Philadelphia County, Philadelphia city,
Ward 7, Enumeration District 412, p. 23B, dwelling 156, family 189,
Johann Mueller household; NARA microfilm T9, roll 1167;
digital images, Ancestry (accessed 15 Dec 2025).
```

### Vital Record (Certificate)

```
[Jurisdiction], [Record type], [date or number],
[person name]; [office holding record];
[access information].
```

**Example**:
```
Bavaria, Birth Register, 1850, entry 127, Johann Mueller;
Standesamt München; certified copy received from
Bavarian State Archives, 10 November 2025.
```

### Newspaper

```
"[Article title or description]," [Newspaper name],
[place of publication], [date], [page/column];
[access information].
```

**Example**:
```
"Mueller-Schmidt Wedding," Münchner Tagblatt (Munich, Bavaria),
5 June 1875, p. 3, col. 2; digital images, Bayerische Staatsbibliothek
(https://www.digitale-sammlungen.de : accessed 15 Dec 2025).
```

### Probate Record

```
[Jurisdiction], [Court name], [Case type], [person name],
[date], [specific locator]; [archive]; [access information].
```

**Example**:
```
Philadelphia County, Pennsylvania, Register of Wills,
Estate of Johann Mueller, 1920, file no. 4523, will dated 15 March 1920;
Philadelphia City Archives; examined 20 November 2025.
```

### Immigration Record

```
[Ship manifest or border record type], [port of arrival], [date],
[ship name (if applicable)], [person entry]; [archive/repository];
[access information].
```

**Example**:
```
New York Passenger Lists, 1820-1957, Port of New York,
arrival 15 May 1875, SS Germania, list 243, line 18,
Johann Mueller; NARA microfilm M237, roll 394;
digital images, Ancestry (accessed 15 Dec 2025).
```

### Military Record

```
[Record type], [name], [service branch/unit]; [repository];
[specific locator]; [access information].
```

**Example**:
```
Civil War Pension File, Johann Mueller, Private, Co. B,
47th Pennsylvania Volunteer Infantry; National Archives,
Washington, D.C., application no. 567890, certificate no. 123456;
copies received 10 December 2025.
```

## Abbreviated Citations

For subsequent references to the same source, use short form:

**Full first citation**:
```
"1880 United States Federal Census," database with images,
Ancestry (https://www.ancestry.com : accessed 15 Dec 2025),
entry for Johann Mueller household, Philadelphia Ward 7,
Philadelphia, Pennsylvania; citing NARA microfilm T9, roll 1167.
```

**Short form for later references**:
```
1880 U.S. Census, Philadelphia, Pennsylvania,
Johann Mueller household (as cited above).
```

## Citation Quality Indicators

### Source Classification

| Type | Definition | Reliability |
|------|------------|-------------|
| **Original** | First recording of event | Highest |
| **Derivative** | Copied or transcribed | Medium |
| **Authored** | Created from analysis | Lowest |

### Information Classification

| Type | Definition | Reliability |
|------|------------|-------------|
| **Primary** | From participant/eyewitness | Highest |
| **Secondary** | Hearsay/secondhand | Medium |
| **Undetermined** | Unknown informant | Variable |

### Evidence Classification

| Type | Definition | Usage |
|------|------------|-------|
| **Direct** | States the fact explicitly | Strong proof |
| **Indirect** | Requires inference | Supporting proof |
| **Negative** | Absence of expected info | Circumstantial |

## Online Database Citations

### FamilySearch

```
"[Collection Name]," database [with images], FamilySearch
(https://www.familysearch.org : accessed [date]),
entry for [name], [specific locator];
citing [original source information].
```

### Ancestry

```
"[Collection Name]," database [with images], Ancestry
(https://www.ancestry.com : accessed [date]),
entry for [name], [specific locator];
citing [original source information].
```

### Findmypast

```
"[Collection Name]," database, Findmypast
(https://www.findmypast.com : accessed [date]),
entry for [name], [specific locator].
```

## Citation for DNA Evidence

```
[Testing Company], [Test Type], [Kit Number/ID];
[Match Name], predicted relationship [relationship],
shared DNA [cM amount]; results downloaded [date].
```

**Example**:
```
Ancestry DNA, autosomal test, kit A1234567;
match Hans Schmidt, predicted 2nd-3rd cousin,
shared DNA 185 cM across 12 segments;
results downloaded 15 December 2025.
```

## Best Practices

1. **Cite every fact** - Even "obvious" facts need sources
2. **Be specific** - Include page, entry, line numbers
3. **Note access dates** - Online content can change
4. **Distinguish image from index** - Index errors are common
5. **Include original source** - Not just the website
6. **Note informant quality** - Who provided the information
7. **Keep copies** - Download images when possible
8. **Update citations** - If URLs change

---

## Quality Checklist

For each citation, verify:

- [ ] Repository/archive identified
- [ ] Collection/record group named
- [ ] Specific document/entry located
- [ ] Date accessed recorded
- [ ] URL provided (if online)
- [ ] Original source cited (not just database)
- [ ] Informant quality noted (if applicable)
- [ ] Copy saved locally (if possible)
